# Protótipo de Baixa Fidelidade - Aplicativo ReCiclo (IHC & UX)


Fernando Ferreira de Oliveira

---

 Proposta de Valor

O aplicativo **ReCiclo** tem como principal objetivo conectar pessoas que desejam doar objetos com outras que precisam desses itens, promovendo a reutilização de materiais, reduzindo o desperdício e incentivando práticas sustentáveis.

---

 Descrição das Telas

Tela 1 - Home / Feed

Apresenta uma lista de itens disponíveis para doação próximos ao usuário. Cada item possui imagem, nome e categoria. O usuário pode clicar em um item para ver mais detalhes.

 Tela 2 - Filtros

Permite ao usuário refinar a busca por itens, utilizando critérios como categoria, distância e estado de conservação.

Tela 3 - Detalhes do Item

Exibe informações completas do item selecionado, incluindo descrição, imagens e dados do doador. Também possui botão para iniciar conversa.

Tela 4 - Cadastro de Item

Formulário onde o usuário pode cadastrar um novo item para doação, informando nome, descrição, categoria e adicionando uma imagem.

Tela 5 - Chat

Tela de conversa entre doador e interessado, permitindo combinar detalhes da retirada do item.

 Tela 6 - Perfil do Usuário

Mostra informações do usuário e suas conquistas, como os "Selos Eco" adquiridos após doações concluídas.

---

Jornada do Usuário

O usuário abre o aplicativo e acessa a tela inicial (Feed), onde visualiza itens disponíveis.
Ele pode utilizar os filtros para encontrar algo específico.
Ao encontrar um item de interesse, acessa a tela de detalhes e inicia uma conversa com o doador.
Após combinar a retirada e concluir a doação, o usuário recebe um "Selo Eco" como forma de incentivo.
